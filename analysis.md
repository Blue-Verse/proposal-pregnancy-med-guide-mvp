■ 프로젝트 분석
임산부/수유부의 약물 복용 안전성 정보를 제공하는 MVP 앱으로, 약사(대표자)의 엑셀 기반 판정 알고리즘을 If-Then 조건부 로직으로 시스템화하는 것이 핵심입니다. 임신 주수·모유 수유 여부에 따라 동일 약물의 위험도가 달라지는 점을 반영하여, 주수별 분기 처리와 예외 조건을 정밀하게 구현합니다. Flutter를 활용하여 iOS/Android 앱 1종으로 양 플랫폼을 커버하고, 관리자 웹(Next.js)으로 약물 DB를 체계적으로 관리합니다.

■ 작업 일정

[Phase 1: 기획/설계] Day 1~14 (2주)
- 대표자(약사)와 엑셀 판정 알고리즘 로직 협의 및 디지털화 설계
- 기능 명세서, If-Then 알고리즘 흐름도, ERD, API 명세서 작성

[Phase 2: UI/UX 디자인] Day 15~28 (2주)
- 임산부 사용자를 고려한 직관적이고 편안한 모바일 앱 디자인
- 관리자 웹 UI 디자인

[Phase 3: 개발] Day 29~72 (약 6주)
- Flutter 모바일 앱 개발 (회원/검색/판정결과/마이페이지)
- NestJS 백엔드 API, If-Then 매칭 알고리즘, 약물 DB 구축
- 엑셀 데이터 일괄 업로드(Data Import) 구현
- 관리자 웹 (회원 관리, 약물 CRUD)
- OCR 약 봉투 인식 (선택, Google ML Kit 활용)

[Phase 4: QA/배포] Day 73~90 (약 2.5주)
